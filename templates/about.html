Introduction¶

QBank is a web application that assists dynamic problem authoring and publishing.

Authors can create dynamic problems by specifying a static question template with variables and values the variables can take. These problems can be exported and displayed in tools that support dynamic questions. QBank also allows you to use existing problems to create new problems or export them directly for use in learning reasources.

Features of QBank¶

The key features of the QBank are:

  1. Separate user interfaces for problem authoring based on the main Problem types:
    • Simple problem – Multiple Choice Question
    • Parameterized problem – Dynamic Question
      • Variable that take a List of values
      • Variable with values that vary over a Range
    • Summative problem – Multi-Part Question
    • Tool Specific problem – Khan Academy Exercise Question
  2. Problem publishing - Parsing options to export and download authored questions to different formats
    • Comma Separated Format - CSV
    • Khan Academy Exercise Format
