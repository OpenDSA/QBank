  • Filename used to save the problem onto database
  • Filename for the problem generated in different formats

Variable name and values
   
Variable Name: The identifier that you can use later in the problem to refer to the variable. Whenever a variable is referred to, the variable name has to be enclosed to <var>...</var>

Variable Value: The values that need to be separated using commas and specified within double quotes.

Eg.:

Variable Name: NO_OF_QUARTERS

Variable Value: "3","4","5","6"

Variable Name: DATA_STRUCTURE

Variable Value: "Stack", "Queue", "Binary Tree"
